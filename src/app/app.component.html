Projects

Chat Server

I have always been interested in real time web applications and web sockets, so as a side project, I decided to try and implement a real time chat server. I am using Angular 4 for the front end, and Node.js for the backend REST API's. I am using Socket.io to implement the real time socket communication between the client and server, so that a user does not have to refresh their page every time a user connects or a message is sent. I am still actively improving this project by adding error handling for duplicate user creations and adding options to delete and edit a profile.
