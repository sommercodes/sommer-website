Chat Server

I have always been interested in real time web applications and web sockets, so as a side project, I decided to try and implement a real time chat server. I used Anguler 4 for the front end, and Node.js for the backend REST API's. I utilized Socket.io to implement the real time socket communication between the client and server, so that a user does not have to refresh their page every time a user connects or a message is sent. To store the messages and users, I am using MongoDB. For user authentication, I am using JWT and I am using the bcrypt library to store hashed user passwords. I am still currently developing this project and making it more robust. I plan to add cases for users to delete accounts, and I am also working on error handling for the cases where users want to create duplicate accounts. Check out the application at the link below!
