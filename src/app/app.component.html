Experience

Software Development Internship - IBM

I am currently finishing up a 16 month software development internship with IBM Security. Throughout my internship, I have had the opportunity to work on web applications using modern web development technologies such as Angular 4, Node.js, and Docker. I worked on front end development, where I implemented views and created reusable common components including a trends graph for data visualizations. I also had the opportunity to work on user authentication using Passportjs and JWT's, as well implementing RESTful API's within our back end microservices which were consumed by our front end. I learned how to set up a continuous integration/development pipeline by using Travis CI to automatically test, build, and deploy our microservices. Overall, this internship has taught me to always be curious and never stop looking for ways to learn and improve myself as a developer.
